# mimamori-flutter

Flutter プロジェクト（iOS / Android / Web 対応）。

## 技術スタック
- **フレームワーク**: Flutter
- **言語**: Dart

## 開発コマンド
```bash
flutter run          # デバッグ実行（接続デバイス/エミュレータ）
flutter build apk    # Android APK ビルド
flutter build ios    # iOS ビルド（macOS のみ）
flutter test         # テスト実行
```

## 注意
- 雛形は `flutter create` で生成されています。
